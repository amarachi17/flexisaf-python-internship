# Week 03 - Decision Making with Python

## Description
This week focuses on applying decision-making concepts in Python using conditional statements and logical operators.  
All tasks are implemented in a single script for simplicity and ease of review

## Tasks Implemented
- Age validation
- Login authentication flow
- Loan eligibility checker
- Expense tracker with overspending alert

## Concepts Covered
- if, else statements
- Comparison operators
- Logical operators
- User input handling

## Implementation Approach
All deliverables are grouped into one file and clearly seperated using section headers.
This approach keeps the code readable and easy to test.

## How to Run
Navigate to the `week-03` directory and run:
python decision_making.py
