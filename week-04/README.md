# Secure Finance Assistant

## Description
This is a beginner-friendly Python console application that focuses on security awareness and personal finance management.

## Features
- Password strength checker with actionable feedback
- Interactive expense tracking system
- Categorized expense summary
- Clean and readable console output

## Technologies Used
- Python 3
- Core programming concepts (loops, conditionals, lists, dictionaries)

## How to Run the Project
Navigate to the `week-04` directory and run:
python secure_finance_assistant.py

## Learning Focus
- Input validation
- Loop control
- Conditional logic
- Data structuring
- Writing clean and resuable code
